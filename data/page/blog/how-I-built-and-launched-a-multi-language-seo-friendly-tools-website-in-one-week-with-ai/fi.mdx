## ⚙️ Teknologiapino: Rakennettu nopeutta ja SEO:ta varten

- **Kehys:** Next.js 15 (App Router)
- **Hosting:** Cloudflare Pages
- **CSS:** Tailwind
- **UI:** Shadcn
- **Tekoäly:** GitHub Copilot + Claude
- **CI/CD:** GitHub → Cloudflare Adapter (`opennextjs-cloudflare`)

Miksi tämä yhdistelmä?

- App Router = sisäänrakennetut layoutit, metatiedot, reititys kielen mukaan
- Cloudflare = edge-toimitettu, nollakustannus, nopea

Ei backendiä, ei tietokantaa, ei palvelinkustannuksia. Vain puhdasta hyötyohjelmien toimitusta.

## ✨ Tekoäly auttoi minua 10x: Näin se tapahtui

> "Tekoäly ei korvaa sinua, se korvaa tylsät osat."

Käytin **GitHub Copilotia** VS Codessa automaattiseen täydennykseen:

- Lomakkeiden syöttö ja tilanhallinta
- Hyötyfunktiot (UUID:t, koodaajat jne.)
- Boilerplate-sivukomponentit
- Blogikirjoitusmallit

Sitten käytin **Claudea** (agenttityyliin) kysymyksiin:

- "Kuinka pakata kuva asiakaspuolella selaimessa?"
- "Mikä on paras tapa jäsentää cron-merkkijonot luettavaksi tekstiksi?"
- "Voitko kääntää tämän i18n-tiedoston japaniksi ja koreaksi?"

Laitoin Clauden jopa kirjoittamaan:

- UKK:t
- Blogikirjoitusten introt
- `meta`-tagimallit 15 kielelle
- JSON-LD-skeemat SEO:ta varten

Copilot = koodin tuottaja
Claude = promptipohjainen avustaja
Yhdessä? Solo-kehittäjän huijauskoodi.

## 🔍 SEO päivästä 1: Rakenne ennen sisältöä

Jos rakennat työkalusivustoa ja **et ajattele SEO:ta**, tuhlaat sen potentiaalia.

Asioita, jotka julkaisin aikaisin:

- ✅ `sitemap.xml` `next-sitemap`:n kautta
- ✅ `robots.txt`
- ✅ JSON-LD rakenteellinen data työkaluille + UKK
- ✅ Sivukohtaiset metatittelit/kuvaukset 15 kielellä (tekoälyn kirjoittamat)
- ✅ Kanoninen + `hreflang`-tagit

> "Luuletko SEO:n olevan hidasta? Kokeile tehdä se _julkaisun jälkeen_."

### Arvokas vinkki:

Prompt tekoälylle:

> "Kirjoita SEO-metakuvaus \[työkalu\]-sivulle \[kielellä\] alle 155 merkkiä."

## 🤔 Luuletko edelleen, ettei sivukartta ole tärkeä?

GoogleBot ei voi indeksoida sitä, mitä se ei löydä.
Työkalusi = näkymättömiä, jos:

- Ei sivukarttaa
- Ei linkkejä
- Ei sisäistä reititystä

Tämän korjaaminen on 1 tunnin työ. Älä ohita sitä.
