# Waarom URL-trackers belangrijk zijn en hoe je ze verwijdert

Als je een link deelt of aanklikt, bevat deze vaak **trackingparameters** in de URL.
Deze _URL-trackers_ (zoals `utm_source` of `fbclid`) worden door bedrijven gebruikt om te volgen waar je vandaan komt, wat je klikt en welke campagne je naar hen bracht.

Handig voor marketeers, maar ze brengen serieuze **privacyrisico's** met zich mee.

---

## 🔒 Hoe URL-trackers je privacy beïnvloeden

- **Je surfgedrag wordt zichtbaar**: Trackers laten zien van welke site of app je komt.
- **Cross-platform profilering**: Adverteerders koppelen je activiteiten op verschillende websites.
- **Onleesbare, lange URLs**: Links vol met parameters zijn moeilijk te lezen.
- **Ongewenste gegevensdeling**: Informatie wordt vaak zonder jouw toestemming naar derden gestuurd.

Kortom: URL-trackers zijn onzichtbare kruimels die je overal op het internet achterlaat.

---

## 📑 Overzicht van veelgebruikte URL-trackingparameters

Dit overzicht bevat de meest gebruikte trackingparameters, hun doel en gebruik. Gebaseerd op standaarden, analyticsplatforms, advertentiesystemen en SEO best practices.

---

### 1. UTM-parameters (Urchin Tracking Module)

Standaard marketingparameters, ondersteund door Google Analytics en veel andere platforms.

| Parameter      | Beschrijving                                                    |
| -------------- | --------------------------------------------------------------- |
| `utm_source`   | Bron van het verkeer (bijv. `google`, `newsletter`, `facebook`) |
| `utm_medium`   | Marketingmedium (bijv. `email`, `cpc`, `social`)                |
| `utm_campaign` | Campagnenaam (bijv. `spring_sale`)                              |
| `utm_term`     | Betaalde zoekwoorden (bijv. `running+shoes`)                    |
| `utm_content`  | Onderscheidt vergelijkbare content/links (bijv. `logolink`)     |

---

### 2. Advertentieklik-ID's

Automatisch gegenereerde parameters door advertentieplatforms voor klik-attributie.

| Parameter | Platform / Doel            |
| --------- | -------------------------- |
| `gclid`   | Google Ads klik-ID         |
| `dclid`   | Google Display Ads klik-ID |
| `fbclid`  | Facebook/Instagram klik-ID |
| `msclkid` | Microsoft/Bing Ads klik-ID |
| `twclid`  | Twitter/X klik-ID          |
| `yclid`   | Yandex Ads klik-ID         |

---

### 3. Referral- & affiliate-tracking

Voor het toewijzen van referrals, partnerschappen of affiliatecampagnes.

| Parameter              | Beschrijving             |
| ---------------------- | ------------------------ |
| `ref` / `referrer`     | Referralbron             |
| `aff_id` / `affiliate` | Affiliate- of partner-ID |
| `partner`              | Partnerprogramma/-bron   |
| `campaignid`           | Campagne-ID              |
| `adgroupid`            | Advertentiegroep-ID      |
| `creative`             | Creatief asset-ID        |

---

### 4. Experimenten / A/B-testen

Parameters om gebruikers in testgroepen te plaatsen.

| Parameter              | Doel                          |
| ---------------------- | ----------------------------- |
| `experiment` / `expid` | Naam of ID van het experiment |
| `variant`              | Variant of testversie         |
| `bucket` / `group`     | Testgroep/bucket-ID           |

---

### 5. Sessie- & clienttracking

Parameters voor sessie- of clientidentificatie (analytics).

| Parameter           | Doel                              |
| ------------------- | --------------------------------- |
| `sessionid` / `sid` | Sessie-ID (tegenwoordig zeldzaam) |
| `cid` / `clientid`  | Client- of gebruikers-ID          |
| `tracking_id`       | Interne tracking-ID               |

---

### 6. Platformspecifieke parameters

Speciale parameters van diverse platforms of tools.

| Parameter          | Platform / Doel          |
| ------------------ | ------------------------ |
| `mc_eid`           | Mailchimp e-mailtracking |
| `_hsenc` / `_hsmi` | HubSpot e-mailtracking   |
| `igshid`           | Instagram share-ID       |

---

### 7. Actieve vs passieve parameters

Of ze de pagina-inhoud beïnvloeden of alleen voor tracking zijn:

- **Actieve parameters**: Veranderen de inhoud of functionaliteit van de pagina.
  Voorbeelden:
- `category=shoes`, `color=blue`, `sort=price_asc`, `page=2`, `lang=de`
- **Passieve parameters**: Alleen voor tracking, geen invloed op de inhoud.
  Voorbeelden:
- `utm_*`, `gclid`, `ref`, `sessionid`
