## ⚙️ Stack: Construída para velocidade e SEO

- **Framework:** Next.js 15 (App Router)
- **Hospedagem:** Cloudflare Pages
- **CSS:** Tailwind
- **UI:** Shadcn
- **IA:** GitHub Copilot + Claude
- **CI/CD:** GitHub → Adaptador Cloudflare (`opennextjs-cloudflare`)

Por que essa combinação?

- App Router = layouts incorporados, metadados, roteamento por localidade
- Cloudflare = implantação na borda, custo zero, rápido

Sem backend, sem BD, sem custos de servidor. Apenas entrega pura de utilitários.

## ✨ A IA me ajudou 10x: Veja como

> "A IA não está substituindo você, está substituindo as partes chatas."

Usei o **GitHub Copilot** dentro do VS Code para autocompletar:

- Entradas de formulário e manipuladores de estado
- Funções utilitárias (UUIDs, codificadores, etc.)
- Componentes de página boilerplate
- Modelos de postagens de blog

Depois usei o **Claude** (estilo agente) para:

- "Como comprimir uma imagem do lado do cliente no navegador?"
- "Qual é a melhor maneira de analisar strings cron em texto legível?"
- "Você pode traduzir este arquivo i18n para japonês e coreano?"

Eu até fiz o Claude escrever:

- FAQs
- Introduções de postagens de blog
- Modelos de tags `meta` para 15 idiomas
- Esquemas JSON-LD para SEO

Copilot = gerador de código
Claude = assistente baseado em prompts
Juntos? Um código de trapaça para desenvolvedores solo.

Meu fluxo de trabalho:

1. Todo o texto em `en.json`
2. Usar Claude para traduzir para cada localidade alvo
3. Revisar manualmente palavras sensíveis ao contexto
4. Verificação automática de chaves ausentes via script

O roteamento `i18n` do App Router + importações dinâmicas tornaram os caminhos localizados limpos.

Bônus de SEO: Adicionei tags `hreflang` + `alternate` para cada rota.

## 🔍 SEO desde o dia 1: Estrutura antes do conteúdo

Se você está construindo um site de ferramentas e **não está pensando em SEO**, está desperdiçando seu potencial.

Coisas que lancei cedo:

- ✅ `sitemap.xml` via `next-sitemap`
- ✅ `robots.txt`
- ✅ Dados estruturados JSON-LD para ferramentas + FAQ
- ✅ Títulos/descrições meta por página em 15 idiomas (escritos por IA)
- ✅ Tags canônicas + `hreflang`

> "Você acha que SEO é lento? Tente fazê-lo _depois_ do lançamento."

### Dica valiosa:

Prompt para IA:

> "Escreva uma meta descrição SEO para uma página de \[ferramenta\] em \[idioma\] com menos de 155 caracteres."

Funciona surpreendentemente bem. Ainda ajusto, mas é 10x mais rápido.
